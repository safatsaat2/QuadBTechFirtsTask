## Overview

The task is to create a Movie Ticket Booking website , I have created this website which can book movie

### The challenge

Users should be able to:

- User should be able to see a Movie Name. :heavy_check_mark:
- User should be able to see Movie Picture on the Card and few details. :heavy_check_mark:
- User can go to other page to see more details about movie. :heavy_check_mark:
- After going to other page user can book movie and it should be store in to localStorage :heavy_check_mark:
